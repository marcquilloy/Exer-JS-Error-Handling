Exer: JS Error Handling

Write a JavaScript function called divideNumbers that takes two parameters: numerator and denominator. Inside the function, use a try block to divide the numerator by the denominator. If the division is successful, return the result. If there's an error (such as division by zero), use a catch block to catch the error and throw a custom error message indicating that the division cannot be performed.
